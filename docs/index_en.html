Getting Started

Follow along step by step to create your first workflow.

1

Create Knowledge Base

Upload documents (PDF, TXT) to create a knowledge base. You can organize them systematically with folders.

KB Creation Screen

File upload, folder creation

2

Add & Connect Nodes

Add nodes using the top buttons and drag handles to connect them. Configure according to your data flow.

Node Add & Connect Demo

Configure Nodes
