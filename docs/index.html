주요 기능

강력한 워크플로우 빌더

시각적 캔버스, 다중 모델 지원, 지식 베이스 관리 등 핵심 기능을 제공합니다.

1

시각적 워크플로우 캔버스

React Flow 기반의 직관적인 캔버스에서 노드를 배치하고 연결합니다.

  • 드래그 앤 드롭으로 노드 배치
  • 노드 간 연결로 데이터 흐름 정의
  • 실행 상태 실시간 시각화
  • JSON 내보내기/가져오기
2

노드별 상세 설정

각 노드를 클릭하면 세부 설정 패널이 열립니다.

  • LLM 프로바이더 및 모델 선택
  • 프롬프트 템플릿 커스터마이징
  • 지식 베이스 선택 및 검색 강도
  • Rerank 옵션 설정
3

지식 베이스 관리

PDF, TXT 문서를 업로드하여 벡터 DB를 생성하고 RAG에 활용합니다.

  • PDF, TXT 파일 업로드
  • 폴더 구조로 체계적 관리
  • BGE-M3 다국어 임베딩
  • 비밀번호 기반 보호
4

실시간 스트리밍 실행

워크플로우 실행 시 LLM 응답이 실시간으로 스트리밍됩니다.

  • LLM 응답 실시간 스트리밍
  • 노드별 실행 상태 표시
  • 병렬 노드 동시 실행
  • 실행 중단 기능

사용법

시작하기

단계별로 따라하며 첫 워크플로우를 만들어보세요.

1

지식 베이스 생성

분석할 문서(PDF, TXT)를 업로드하여 지식 베이스를 생성합니다. 폴더를 만들어 체계적으로 관리할 수 있습니다.

2

노드 추가 및 연결

상단 버튼으로 노드를 추가하고, 핸들을 드래그하여 노드를 연결합니다. 여러 생성 노드의 결과를 앙상블 노드로 통합하고, 검증 노드를 통해 LLM 출력을 지식 베이스와 대조 검증하여 할루시네이션을 줄일 수 있습니다.

3

노드 설정

각 노드를 클릭하여 LLM Provider와 모델 선택, 출력 형식 설정, 프롬프트 입력을 할 수 있습니다. 프롬프트에서 연결된 이전 노드의 출력은 {input_data}로, 연결된 컨텍스트 노드의 검색 결과는 {context} 변수로 받아 활용할 수 있습니다.

4

워크플로우 실행

"워크플로우 실행" 버튼을 클릭하면 노드가 순차적/병렬로 실행되며, 결과가 실시간으로 스트리밍됩니다.

5

결과 확인 및 저장

출력 노드와 결과 패널에서 최종 결과를 확인합니다. 워크플로우를 저장하거나 JSON으로 내보내 재사용하세요.
